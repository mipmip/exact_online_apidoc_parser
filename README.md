# ExactOnlineApidocParser

Parses Exact Online Api Reference

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'exact_online_apidoc_parser'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install exact_online_apidoc_parser
